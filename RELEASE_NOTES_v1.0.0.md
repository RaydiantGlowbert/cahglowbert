# Cards Against Humanity v1.0.0

Release date: 2026-07-02
Tag: v1.0.0

## Summary

This release marks the official completion of the online multiplayer game.

## Included in v1.0.0

- Online room-based multiplayer flow
- Server-authoritative game lifecycle
- Simultaneous non-judge card submissions
- Judge winner selection from anonymized submissions
- Host-controlled round progression
- Session reconnection support
- Production deployment on Vercel (frontend) and Render (backend)

## Product Direction Finalized for v1

- Local mode removed from the app shell
- Remote-only experience in the main UI
- Launch copy and labels polished for public release
